# Concept 06: Lists & Keys in React

### Core Learnings:
- **Array Mapping (`.map()`)**: Utilized the JavaScript `.map()` method to dynamically loop through arrays and transform raw data into JSX elements.
- **The Role of Keys**: Implemented unique `key={todo.id}` identifiers on list items, allowing React to optimize virtual DOM reconciliations efficiently.
- **Spread Operator (`[...]`)**: Practiced mutating state arrays immutably by copying previous state values and appending new items.
