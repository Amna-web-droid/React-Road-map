# Concept 02: Props in React

### Core Learnings:
- **Props (Properties)** are inputs passed from Parent to Child components.
- They are **Read-Only** (Immutable).
- **Destructuring** makes the code cleaner by unpacking variables from props directly in the argument list.
- **Default Props** act as backup values when parents forget to pass attributes.

### Real-World Project:
- **Skill Cards Generator**: Used reusable `<SkillCard />` components inside a grid list, passing dynamic colors, levels, titles, and experiences as Props.
