# Concept 05: Conditional Rendering in React

### Core Learnings:
- **Ternary Operator (`? :`)**: Used for double-sided conditions (if-else style rendering) directly within JSX.
- **Logical AND (`&&`)**: Ideal for rendering elements optionally (if true, render; if false, do nothing) to keep the markup clean.
- **Dynamic Elements**: Swapped UI cards, badges, states, and buttons seamlessly based on a single boolean state (`isLoggedIn`).

### Project:
- **Login Status Toggle**: A dashboard prototype that restricts or displays proprietary elements depending on user authentication status.
