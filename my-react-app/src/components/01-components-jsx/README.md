# Concept 01: Components & JSX

### Key Takeaways:
1. **Components** are reusable JavaScript functions that return UI. Their names must start with a Capital letter.
2. **JSX** is a syntax extension for JavaScript that looks like HTML but executes as JS.
3. **JSX Rules**:
   - Must return a single parent element (or use React Fragment `<> </>`).
   - Use `className` instead of `class`.
   - Self-closing tags like `<img>` must end with a slash: `<img />`.
   - Inline styles are passed as JavaScript objects using camelCase properties.
